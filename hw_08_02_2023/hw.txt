1. git clone "ссылка на репозиторий напр: git@github.com:SmirnovAlex0891/GIT-java_43_45m-qa_44_46m.git"
можно предварительно создать папку в которую будет склонирован репозиторий, но не обязательно т.к.
репозиторий сам будет склонирован как папка в то место где будет выполнека комманда git clone.

2. перейти в каталог который должен стать репозиторием,
выполнить git init,
потом добавить что нужно в отслеживание git add .(добавить все например),
потом закоммитить то что в отслеживании git commit -m 'описание коммита',
потом связать его с репозиторием на git hub
например: git remote add origin git@github.com:SmirnovAlex0891/GIT-java_43_45m-qa_44_46m.git,
и если это первый раз то: git push -u origin master (или main), в последующем: git push

3. получить изменения: git pull

4. git status -- состояние файлов и каталогов: отслеживается/не отслеживается, 
измененный/не измененный, добавлен в отслеживание/не добавлен, закомичен/не закомичен;
git log -- список коммитов;
git remote -v -- выводит инф. о связке локального репозитория с удаленным репозиторием.
